# 银河猎手 — AI 生图提示词手册(v0.10.2)

> 用即梦 / 豆包(字节系,中文提示词效果最佳)生成雷电/怒首领蜂级的机械硬表面游戏资产。
> 本文严格遵循《太空射击游戏美术规范 v1.0》。生成后导出 PNG 丢到 `src/assets/sprites/_raw/`,
> 跑 `node tools/strip-bg.js` 自动抠背景,即可在游戏中生效(渐进增强,缺图自动退回程序化)。

---

## 0. 生图要点(必读)

1. **用即梦或豆包**,中文提示词。优先即梦的「图像生成」/「参考图」模式。
2. **背景必须是纯黑** `#000000`(便于抠图脚本去背景)。提示词里已写"纯黑背景"。
3. **飞船机头朝正上方**(竖版射击,玩家在下方往上打)。
4. **每张单独生成**,不要拼成图集——脚本按文件名处理。
5. **导出尺寸**尽量 ≥ 512px,方形或竖向均可。
6. **文件名严格按下方清单**(脚本按文件名认图)。

```

必须包含五层结构:①外装甲板(硬表面)②内部骨架结构 ③小型能量核心(发光但≤20%,仅核心/引擎处)④推进器喷口 ⑤武器炮口/枪口。
结构占80%,光效不超过20%,光只能强调核心与引擎,绝不能用光替代形状。
```

## 4. 操作流程

1. 打开即梦/豆包,用上方提示词,**逐张生成**(每张单独出图)。
2. 每张导出 PNG,按文件名存到:
   ```
   C:\Users\xhxq1024\Desktop\GalacticHunter\src\assets\sprites\_raw\
   ```
   (ship1.png、ship2.png ... enemy-scout.png 等)
3. 告诉我"生好了",我跑:
   ```
   node tools/strip-bg.js
   ```
   自动去背景(抠纯黑)→ 输出到 `src/assets/sprites/`。
4. 贴图在游戏中即时生效(`render.js` 已接好)。你打开游戏实机看效果,我按你反馈调光晕/大小/朝向。

## 5. 抠图说明(strip-bg.js)

- 自动采样四角颜色判断背景色(不限于纯黑,即梦给深色背景也能抠)。
- 容差范围内判为背景→透明,边缘羽化避免锯齿。
- 若即梦背景不是纯黑(如自带星空),生成时务必让飞船周围有大片均匀深色留白,保证四角采样到背景色。
- 抠不干净告诉我背景是什么色,我调容差。
