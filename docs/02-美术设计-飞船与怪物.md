# 银河猎手 — 美术系统设计(飞船与怪物)

| 项 | 内容 |
|---|---|
| 文档版本 | v0.6(2026-06-25) |
| 定位 | 2D / 伪 3D 程序化 Canvas 美术系统,支持「升级外观变化 + 多敌人类型」 |
| 实现 | `src/js/render.js`(运行时按等级/类型组装模块,非静态贴图) |

> **为什么是程序化 Canvas 而非 PNG 贴图**:升级外观变化要求飞船随等级动态变形(机翼展开、推进器增加、能量核心增强),静态 PNG 无法表达。Canvas2D 的渐变 / 合成 / 路径远强于扫描线光栅化,且不依赖任何二进制图片资源 —— 沿用项目「避开图像生成/识别报错」的哲学。`assets.js` / `gen-sprites.js` 保留不破坏既有工作流,但 render 层已改用更强的程序化绘制为主路径。

## 一、玩家飞船(模块化 + 阶段进化)

飞船由 5 个模块组成,按 `ship.level`(1–5)逐级组装。所有模块在 `render.js` 的 `ship()` 中以本地坐标(原点=飞船中心,旋转 `aimAngle`)绘制,统一乘 `visScale`(体型随等级放大,碰撞半径不变)。

### 1.1 模块拆分(结构建议)

| 模块 | 方法 | 作用 | 进化规则 |
|---|---|---|---|
| 推进器 Engine | `_shipEngine` | 喷口 + 尾焰 | 喷口数 1→2→3→4→5,尾焰随等级变长变亮 |
| 机翼 Wings | `_shipWings` | 左右机翼 | 窄→宽→分叉(Lv3+)→能量翼缘(Lv4+) |
| 船体 Hull | `_shipHull` | 中心船体骨架 | 梭形→装甲条纹(Lv2+)→分体结构(Lv4+) |
| 武器挂载 Weapons | `_shipWeapons` | 炮管 | 1→2→3→4 管,高级多联装(Lv3+) |
| 能量核心 Core | `_shipCore` | 中心发光体(**最关键**) | 尺寸/亮度随等级增强,Lv5 悬浮 + 能量环 |

绘制顺序(后到前):引擎 → 机翼 → 船体 → 武器 → 能量核心。

### 1.2 各等级外观描述(可用于建模 / AI 绘图参考)

> 配色:船体金属 `#2a3340`→`#aebccf` 渐变;能量核心与光晕用 `Config.SHIPS[].glow`(Lv1 青→Lv5 品红)。

**Lv1 侦察艇 Scout**
- 小型梭形船体,单推进器,弱发光核心(青色,小而暗)
- 窄短机翼,1 根炮管
- 轻量金属质感,结构单一
- 视觉关键词:轻、快、入门

**Lv2 驱逐舰 Frigate**
- 机翼展开(更宽),增加副推进器(双喷口)
- 船体增加护甲板(横向条纹)
- 武器 1→2,核心变亮
- 视觉关键词:改装、战斗艇

**Lv3 巡洋舰 Cruiser**
- 明显「装甲块拼接结构」(多条分割线)
- 机翼分叉(双层/分裂),3 推进器
- 能量核心外露、发光增强,3 炮管多联装
- 视觉关键词:重型、突击

**Lv4 战列舰 Battleship**
- 船体变大 + 分体结构(中线缝隙)
- 4 推进器,机翼带能量翼缘(发光边)
- 核心强发光,4 炮管
- 视觉关键词:旗舰级、危险

**Lv5 旗舰 Dreadnought**(终极形态)
- 船体最大 + 分体结构,5 推进器环形布局
- 能量核心**悬浮在中心**(脉动浮动)+ 能量环
- 能量翼缘光效最强,4 炮管多联装
- 视觉关键词:母舰、终极、科技感拉满

### 1.3 升级表现规则(强制)

升级必须「看起来变强」,不能只换皮:
1. **体积逐级增加**(`visScale = 1 + (lvl-1)×0.06`)
2. **武器数量增加**(1→2→3→4)
3. **发光强度增强** —— 核心最关键(尺寸 + alpha 随等级增长)
4. **结构 单体 → 拼装 → 机械复杂化**(条纹→分体→悬浮)
5. **推进器尾焰逐级变长**(喷口数 + 焰长)
6. 整体越来越「危险 / 科技感」

## 二、怪物设计系统(多类型生态)

6 种 tier,每种独立轮廓 + 发光弱点核心,一眼可辨类型;复杂度 / 光效随 tier 递增;统一「太空机械 / 生物机械混合」风格。配色见 `Config.ALIENS[].color`。

### 2.1 统一规则
- 都有**发光弱点核心**(中心径向亮光 + `drawGlow` 光晕;Boss 为多核心节点)
- 形状**一眼区分**类型(水滴 / 三角翼 / 六边形 / 波浪体 / 八边形 / 巨构团块)
- 越强 → 结构越复杂 + 光效越强
- 太空机械 / 生物机械混合风格

### 2.2 各类型设计说明

| Tier | 类型 | 轮廓 | 特征 | 玩法定位 |
|---|---|---|---|---|
| t1 | 追踪虫 Drone Bug | 装甲水滴体 | 三尖刺腿 + 上颚 + 小亮核 | 快速 / 自爆感,大量刷新 |
| t2 | 拦截机 Interceptor | 后掠三角翼 | 双引擎光 + 中心核 | 中距射击型,常见 |
| t3 | 装甲巨兽 Armored Brute | 六边形装甲 | 双钳臂 + 凹陷核心 | 高防肉盾,慢 |
| t4 | 相位幽影 Phase Wraith | 半透明波浪体 | 触须 + 闪现回避 | 闪避型(受击概率闪现) |
| t5 | 精英装甲 Elite Armor | 不规则八边形机械体 | 肩炮 + 装甲分割 + 暴露弱点核 | 高分目标,大 |
| t6 | 巨构 Boss Colossus | 分段团块(占屏 1/3) | 多核心节点 + 顶刺 + 装甲段 | 稀有,击杀成就感 |

### 2.3 实现

- `alien()`:外光晕 → 径向装甲渐变本体 → `switch(tier)` 调对应轮廓方法 → 发光弱点核心(非 Boss)→ 状态叠加(冰冻/灼烧)→ 血条
- 各 tier 轮廓方法:`_alienCrawler` / `_alienDrone` / `_alienBrute` / `_alienWraith` / `_alienElite` / `_alienBoss`

## 三、整体美术风格

- **科幻 + 发光能量风**:深空背景 + 高对比霓虹光(青/品红/金/橙)
- **轮廓清晰**:适合手游小屏辨识
- **Arcade 射击风格**(雷电 + 现代科幻)
- **性能**:发光用预渲染精灵(`drawGlow`,按色缓存复用),避免 `shadowBlur`(Canvas2D 头号性能杀手)
- **受击反馈**:本体闪白(`hitFlash`)+ 冰冻冷光圈 / 灼烧橙光脉动(`_alienStatus`)

## 四、扩展建议

- 新增怪物类型:在 `Config.ALIENS` 加 tier,`render.js` `switch` 加对应 `_alienXxx` 方法
- 飞船新等级:`Config.SHIPS` 加级 + `Config.MAX_SHIP_LEVEL` 调高;模块方法的等级参数数组(如 `[0.55,0.85,1.1,1.35,1.6]`)按需扩展
- 若需更高精度:可后续用 AI 生图替换某 tier 的程序化绘制(`render.js` 重新接 `_sprite` 贴图路径),但会牺牲「升级外观变化」的运行时灵活性
