# 银河猎手 — 23 种敌人 AI 生图清单(v0.10.3)

> 全部 23 怪各一独立贴图(零染色、质量最高)。逐张用即梦生成,导出 PNG(文件名 = 怪物编号),
> 存到桌面 `GH-生图丢这里` 文件夹,跑 `node tools/strip-bg.js` 抠图,游戏中即生效(缺失自动退回程序化)。
>
> **生成要点(每张都要):**
> 1. 即梦,中文提示词
> 2. **纯黑背景 #000000**(抠图靠这个)
> 3. **机头朝正上方**(代码会统一翻转 180° 让它朝下,你只管朝上生成)
> 4. 文件名严格按编号:`t1.png`、`t2.png` ... `t23.png`
> 5. 参考雷电/怒首领蜂敌机,机械硬表面,结构 80% / 光效 ≤20%

## 操作流程

1. 即梦按提示词逐张生成,**文件名严格用怪物编号**(t1.png、t2.png ...)
2. 导出存到桌面 `GH-生图丢这里` 文件夹
3. 一批生完(比如第1批 t1-t4),跟我说"**第1批好了**"
4. 我跑 `node tools/strip-bg.js` 抠图 → 游戏中即生效
5. 你实机看效果,我按反馈调

> **不用一次生全 23 张。** 分批来,第 1 批(t1-t4)做好你看了满意,再继续。每批接入后实机验证。
> 飞船已经验证管线可用,敌人走同样流程。
