# Galactic Hunter
银河猎手 — 太空射击休闲小游戏(类捕鱼达人的太空化改造)

纯 HTML5 Canvas + 原生 JS,零构建、零依赖,浏览器双击 `index.html` 即可运行,也适合部署为静态网站 / 移植微信·抖音小游戏。

## 玩法
- 移动鼠标 / 手指:飞船固定在底部、左右移动跟随指针,炮口自动锁定最近敌人 —— 走位闪避怪物下压
- 点击或按住开火;击杀外星怪物获得积分与金币
- 用金币在局内升级三养成线:武器(Lv1→Lv5)、船舰(Lv1→Lv5,放大武器伤害 + 飞船外观随等级模块化进化)、防御(Lv1→Lv5,充能护盾 + 自动回复 / 复活)
- 场上定时掉落技能胶囊,子弹击中即拾取并持久生效直到下一个:冰冻减速 / 火焰(伤害↑+灼烧)/ 闪电连锁 / 激光贯穿,以及双发·三发·四发散射

## 运行
```bash
# 方式一:直接打开
index.html

# 方式二:本地起服务器(推荐,避免 file:// 限制)
npx serve .
# 或 python -m http.server 8080
```

## 冒烟测试
```bash
node tools/smoke.js
```

## 架构(为小游戏移植预留)
```
platform.js  平台适配层(画布/输入/存储)  ← 移植时只改这一层
engine.js    游戏框架(主循环/数学/碰撞/对象池)
render.js    渲染层(写实科幻风,预渲染发光精灵优化性能)
entities.js  实体层(飞船/子弹/怪物/粒子/金币/飘字)
game.js      游戏逻辑(状态机/波次/经济/升级/存档/UI)
config.js    数值配置表(单一数据源)
```
> 上层逻辑不直接依赖 DOM,移植到微信/抖音小游戏时仅需替换 `platform.js`。
